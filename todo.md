### 功能

- [ ] 做一个参数类方便传递
- [ ] 在界面上控制门槛参数
- [ ] 黑边参数
- [ ] ctrl + z 撤销

### 表格

- [ ] 对于第三列以后的单元格，按 delete 表示删除当前光标所在位置的下一个字符。如果光标在最后一个字符后面，那么将下一行单元格的内容拼接到当前单元格，后续单元格的内容依次前移。
- [ ] 当前的第三列及以后的列按 delete，下一行内容拼接还是不起作用。可以研究，当 delete 之后当前单元格内容没变的话，说明光标在最后一个字符后面，那么将下一行单元格的内容拼接到当前单元格光标后，后续单元格的内容依次前移。

- [ ] 如果已经是最后一行，按 Enter 表示在该行下方插入新行。
- [ ] 对于第三列及以后的列，按 shift+delete 表示删除当前列。
- [ ] 允许插入列和删除列，但是仅对第三列以后有效。

- [ ] 支持从 tsv 文件导入数据
